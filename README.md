This Node.js app _securely_ authenticates to your Neurosity headset and maps mental commands to computer shortcuts like "Close other tabs in Chrome".

## Getting started

Run in the terminal:

1. Clone this repo and run `npm install`
2. Turn on your Neurosity headset
3. Run the following command in the terminal:

```
node index.js --email=NEUROSITY_EMAIL --password=NEUROSITY_PASSWORD
```
